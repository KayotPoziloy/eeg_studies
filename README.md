# EEG Studies

Проект по анализу электроэнцефалографических сигналов (ЭЭГ): от самостоятельного сбора данных до цифровой обработки, визуализации и интерпретации мозговых ритмов в разных экспериментальных условиях.

Проект показывает навыки работы с реальными физиологическими данными: шумными, неоднородными, зависящими от качества оборудования и методики эксперимента. В работе сделан акцент не только на построении графиков, но и на проверке гипотез, сравнении условий и честной интерпретации ограничений измерений.

## Что сделано

- Проведены серии 5-минутных ЭЭГ-замеров в разных состояниях: покой, закрытые/открытые глаза, музыка, письмо, печать, решение задач, шахматы, компьютерная игра, зрительный поиск и просмотр видео.
- Реализована загрузка и первичная проверка данных из `.csv` и `.dat` файлов.
- Выполнена предобработка сигналов: нормализация времени, проверка пропусков, переименование каналов, выделение нужных частотных диапазонов.
- Применены полосовые фильтры Баттерворта для выделения ритмов.
- Построены временные графики, спектры мощности и сравнительные диаграммы по экспериментальным условиям.
- Рассчитаны количественные метрики: средняя мощность ритма, пиковая мощность, частота пика, RMS-амплитуда, доля ритма в спектре, межполушарная асимметрия и ERD-подавление для мю-ритма.
- Сформулированы выводы по каждому эксперименту с учетом артефактов, ограничений оборудования и возможных методологических ошибок.

## Технический стек

- Python
- Jupyter Notebook / Google Colab
- Pandas
- NumPy
- Matplotlib
- SciPy
- Цифровая обработка сигналов: Butterworth band-pass filtering, `filtfilt`, Welch PSD, Hilbert transform
- Описательная статистика и сравнительный анализ экспериментальных условий

## Методы анализа

В проекте используются подходы, характерные для задач biomedical data analysis и signal processing:

- выделение частотных диапазонов ЭЭГ-ритмов;
- спектральная плотность мощности методом Уэлча;
- расчет мощности в заданном диапазоне частот;
- сравнение условий эксперимента по численным метрикам;
- анализ межполушарной асимметрии;
- оценка подавления ритма относительно состояния покоя;
- визуальная проверка качества сигнала и фильтрации.

Этот репозиторий демонстрирует не только владение Python-библиотеками, но и умение работать с исследовательской неопределенностью:

## Как запустить

1. Откройте нужный ноутбук в Jupyter Notebook, JupyterLab или Google Colab.
2. Подключите исходные `.csv`/`.dat` файлы с ЭЭГ-измерениями. В ноутбуках используются пути Google Drive, поэтому при локальном запуске пути к данным нужно заменить на свои.
3. Последовательно выполните ячейки: загрузка данных, предобработка, фильтрация, визуализация, расчет метрик и интерпретация.

Минимальный набор зависимостей:

```bash
pip install pandas numpy matplotlib scipy jupyter
```
